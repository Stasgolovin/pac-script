# PAC Script for RU Proxy Bypass

PAC-скрипт для обхода блокировок Роскомнадзора через прокси.

## Использование

URL: `https://bot.curka.ru/proxy.pac`

### Настройка в браузере

1. Откройте настройки прокси
2. Выберите "Автоматическая настройка прокси"
3. Введите URL: `https://bot.curka.ru/proxy.pac`

## Маршрутизация

Через прокси `89.125.213.6:3128`:

- Telegram (домены + IP-диапазоны)
- WhatsApp
- YouTube + CDN
- OpenAI / ChatGPT
- Anthropic / Claude

- Cursor AI

- npmjs / PyPI

- 2ip.ru

Всё остальное — напрямую (DIRECT).
